Bitcoin Forecast Using Sentimental Analysis and Bayesian Network

Abstract- Social media is prominent because of its interactive nature and user-generated content and has always been an excellent tool for digital relationships sharing thoughts and emotions beyond the realm of recommendation to influence marketing.
Bitcoin is a decentralized digital currency, which is, in fact, a revolutionary concept and now attracting much of business and media attention attention because it doesn’t rely on a government or trusted financial institution.

This project analyzes the integration of Sentimental Analysis and Bayesian Network to predict the Bitcoin price trends over public opinion and sentiment.

Scalable video compressing is used to provide scalability to video bit stream. To adapt with the data trasmission rate, the server is required to be able to recognize the network conditions and cope up accordingly.
Fuzzy logic has a fuzzy nature of inputs, which enables it to map the linguistic terms of the problem to the system and large number of variables involved.

This project investigates the implementation of Fuzzy Logic to govern the application's sending rate while satisfying user needs and not wasting the network resources
